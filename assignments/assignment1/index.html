Acronym App (P3 - Assignment 1)

Overview of the Acronym App

The purpose of this assignment is to give you experience using Android to develop an Acronym app using the Retrofit library and an Android Content Provider and SQLite database. Retrofit is used to obtain expansions of an acronym from an Acronym web service. This data is then stored in a timeout cache that uses an SQLite database-backed Content Provider for storage. The user will provide an acronym as a String (such as "BBC") and a List of AcronymData will be returned to the AcronymActivity, which will display the expanded acronym data in a ListView.

Skeleton Code

Skeleton code for this assignment is available . Pull the appropriate skeleton code into your repository, read it carefully, and complete the "TODO" markers, which reside in the following Java source files: AcronymOps.java, AcronymProvider.java, DeleteCacheReceiver.java, and ContentProviderTimeoutCache.java.

Note that there are no automated unit tests for this assignment. Instead, you'll know the program is working when you can successfully access, store, and display the acronym expansions from the Acronym web service. However, make sure your program works for both valid and invalid input.
